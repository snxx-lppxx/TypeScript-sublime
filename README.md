How to use Git?
---
```git-bash
	$ git clone https://github.com/your-username/TypeScript-sublime.git
	cd TypeScript-sublime
	$ git remote add upstream https://github.com/snxx-lppxx/TypeScript-sublime.git
	$ git fetch upstream
```
